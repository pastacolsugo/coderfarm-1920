# Distanze

Scrivere un programma che legga `N` coppie di valori, che rappresentano rispettivamente le coordinate `x` e `y` di `N` punti su un piano.

Per ogni punto, calcolare la distanza di questo punto dall'origine `O (0, 0)`

### Input

```
5
1 1
2 3
4 7
5 2
3 3
```

### Output
```
1.41421
3.60555
8.06226
5.38516
4.24264
```
